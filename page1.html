CSS Syntax

Selectors:

element { property: value; }

#id { property: value; }

.class { property: value; }

element, element { property: value; } (Group Selector)

element element { property: value; } (Descendant Selector)

element > element { property: value; } (Child Selector)

Properties:

color: red;

background-color: #ffffff;

font-size: 16px;

margin: 20px;

padding: 10px;

border: 1px solid black;

Pseudo-classes:

element:hover { property: value; }

element:focus { property: value; }

element:active { property: value; }

Pseudo-elements:

element::before { content: "text"; }

element::after { content: "text"; }

element::first-letter { property: value; }

element::first-line { property: value; }

Common Errors and Solutions

Missing Semicolon:

CSS
color: red /* Missing semicolon */

Solution: Ensure each property-value pair ends with a semicolon.
CSS
color: red;

Incorrect Selector:

CSS
button .submit { color: blue; } /* Incorrect selector */

Solution: Use the correct selector for the intended element.
CSS
.submit { color: blue; }

Typographical Errors in Property Names:

CSS
background-color: red; /* Typo in property name */

Solution: Double-check property names for typos.
CSS
background-color: red;

Incorrect Hex Code:

CSS
color: #ggg; /* Invalid hex code */

Solution: Use valid hex codes.
CSS
color: #fff;

Conflicting Styles:

CSS
.header { color: red; }
.header { color: blue; } /* Conflicting styles */

Solution: Combine or choose one style.
CSS
.header { color: blue; }
